## 环信 WebIM sdk

测试环信WebIM请访问 https://webim.easemob.com。

登录可以使用环信UIDemo的账号和密码（见http://www.easemob.com/demo)

环信WebIM快速入门文档请见：https://github.com/easemob/web-im/blob/master/sdk/quickstart.md

更多关于环信的开发文档请见：https://docs.easemob.com
